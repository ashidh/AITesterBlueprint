# Feature Context: VWO Login Page

**URL:** https://app.vwo.com/
**Description:** The gateway interface for digital marketers, CRO specialists, and stakeholders accessing the Visual Website Optimizer platform.

## Elements & Selectors (Fact-Based)
* **Email Input Field:** Text entry designated for account username.
* **Password Input Field:** Masked text entry for account passcode.
* **Remember Me Checkbox:** Checkbox to configure cookie persistent sessions.
* **Sign In Button:** Form submission control triggering credentials validation.
* **Forgot Password Link:** Redirects user to password reset recovery path.
* **Start a Free Trial Link:** Redirects non-registered visitors to registration portal.
